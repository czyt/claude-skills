# 【小狼毫】輸入法

[Permalink: 【小狼毫】輸入法](https://github.com/rime/weasel/blob/master/README.md#%E5%B0%8F%E7%8B%BC%E6%AF%AB%E8%BC%B8%E5%85%A5%E6%B3%95)

基於 中州韻輸入法引擎／Rime Input Method Engine 等開源技術

式恕堂 版權所無

[![Download](https://camo.githubusercontent.com/941f99a5f2ac8d839fdcf2a5249d64cf44f617081fda71524d9aa68c70940afc/68747470733a2f2f696d672e736869656c64732e696f2f6769746875622f762f72656c656173652f72696d652f77656173656c)](https://github.com/rime/weasel/releases/latest)[![Build status](https://github.com/rime/weasel/actions/workflows/commit-ci.yml/badge.svg)](https://github.com/rime/weasel/actions/workflows/commit-ci.yml)[![GitHub Tag](https://camo.githubusercontent.com/30ceb7a4f73a35c675e3275f4ff3a5be5a457a5d10402dfa27bcaaefac1b8a3f/68747470733a2f2f696d672e736869656c64732e696f2f6769746875622f7461672f72696d652f77656173656c2e737667)](https://github.com/rime/weasel)

授權條款：GPLv3

項目主頁： [https://rime.im](https://rime.im/)

您可能還需要 RIME 用於其他操作系統的發行版：

- ibus-rime、fcitx5-rime 或 fcitx-rime 用於 Linux
- 【鼠鬚管】用於 macOS （64位）

## 安裝輸入法

[Permalink: 安裝輸入法](https://github.com/rime/weasel/blob/master/README.md#%E5%AE%89%E8%A3%9D%E8%BC%B8%E5%85%A5%E6%B3%95)

本品適用於 Windows 8.1 ~ Windows 11

初次安裝時，安裝程序將顯示「安裝選項」對話框。

若要將【小狼毫】註冊到繁體中文（臺灣）鍵盤佈局，請在「輸入語言」欄選擇「中文（臺灣）」，再點擊「安裝」按鈕。

安裝完成後，仍可由開始菜單打開「安裝選項」更改輸入語言。

## 使用輸入法

[Permalink: 使用輸入法](https://github.com/rime/weasel/blob/master/README.md#%E4%BD%BF%E7%94%A8%E8%BC%B8%E5%85%A5%E6%B3%95)

選取輸入法指示器菜單裏的【中】字樣圖標，開始用小狼毫寫字。

可通過快捷鍵 ``Ctrl+` `` 或 `F4` 呼出方案選單、切換輸入方式。

## 定製輸入法

[Permalink: 定製輸入法](https://github.com/rime/weasel/blob/master/README.md#%E5%AE%9A%E8%A3%BD%E8%BC%B8%E5%85%A5%E6%B3%95)

通過 開始菜單 » 小狼毫輸入法 訪問設定工具及常用位置。

用戶詞庫、配置文件位於 `%AppData%\Rime`，可通過菜單中的「用戶文件夾」打開。高水平玩家調教 Rime 輸入法常會用到。

修改詞庫、配置文件後，須「重新部署」方可生效。

定製 Rime 的方法，請參考 Wiki [《定製指南》](https://github.com/rime/home/wiki/CustomizationGuide)。如需定製 Weasel 獨有的樣式和行為，請參考本倉庫 [Wiki 頁面](https://github.com/rime/weasel/wiki)。
