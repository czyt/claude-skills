## 安裝輸入法

[Permalink: 安裝輸入法](https://github.com/rime/squirrel/blob/master/README.md#%E5%AE%89%E8%A3%9D%E8%BC%B8%E5%85%A5%E6%B3%95)

本品適用於 macOS 13.0+

初次安裝，如果在部份應用程序中打不出字，請註銷並重新登錄。

## 使用輸入法

[Permalink: 使用輸入法](https://github.com/rime/squirrel/blob/master/README.md#%E4%BD%BF%E7%94%A8%E8%BC%B8%E5%85%A5%E6%B3%95)

選取輸入法指示器菜單裏的【ㄓ】字樣圖標，開始用鼠鬚管寫字。
通過快捷鍵 ``Ctrl+``` 或 `F4` 呼出方案選單、切換輸入方式。

## 定製輸入法

[Permalink: 定製輸入法](https://github.com/rime/squirrel/blob/master/README.md#%E5%AE%9A%E8%A3%BD%E8%BC%B8%E5%85%A5%E6%B3%95)

定製方法，請參考線上 [幫助文檔](https://rime.im/docs/)。

使用系統輸入法菜單：

- 選中「在線文檔」可打開以上網址
- 編輯用戶設定後，選擇「重新部署」以令修改生效
